As part of my Vinternship at VLED Lab, I built a complete full-stack web application using the MERN stack (MongoDB, Express.js, React.js, and Node.js). This project helped me understand how modern web applications are designed, developed, and deployed from scratch.
In this vlog, I walk through my learning journey — from setting up the backend server and designing RESTful APIs to building a responsive frontend using React. I also demonstrate how I connected the frontend with the backend, managed state, handled authentication, and performed CRUD operations using MongoDB.
This experience strengthened my understanding of full-stack architecture, database design, API integration, and real-world debugging. I also share the challenges I faced, how I solved them, and the key lessons I learned throughout the development process.
This vlog reflects my practical exposure, hands-on coding experience, and growth as a developer during my internship at VLED Lab.
